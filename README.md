# 🤖 AI Study Buddy — Ai-Hackathon

> A Next.js hackathon project featuring a Study Buddy landing page and an AI-powered chat interface for students.

## 📌 About

This project was built during a hackathon. It includes:
- A modern **Study Buddy landing page** with glassmorphism UI
- An **AI-powered chat interface** for students to ask questions
- Built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**

---

## 🚀 Features

- Responsive landing page with smooth UI
- AI chat integration for student queries
- Clean and modern design
- Component-based Next.js architecture

## 💻 Getting Started

```bash
# Clone the repo
git clone https://github.com/Uttkarshchambiyal/Ai-hackathon.git

# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
